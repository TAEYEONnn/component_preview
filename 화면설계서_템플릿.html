공통
  • [서브 헤더]는 화면 최상단에 고정 노출한다.
  • [뒤로가기] 버튼 클릭 시 이전 화면으로 이동한다.
  • [알림] 버튼 클릭 시 알림 목록 페이지로 이동한다.
  • [전체메뉴] 아이콘 클릭 시 우측에서 슬라이드 형태로 전체메뉴 패널이 노출된다.
1
자주 쓰는 서비스
  • 수강생 로그인 시 2열 그리드 아이콘 형태로 노출한다.
  • 게스트/비수강생 시 기본 추천 서비스 5종을 노출한다.
  • 아이콘 클릭 시 해당 서비스 페이지로 이동한다.
1-1 서비스 편집
  • [편집] 버튼 클릭 시 서비스 편집 모드로 진입한다.
  • 드래그 앤 드롭으로 순서를 변경할 수 있다.
  • 최대 6개 등록 가능하며 초과 시 "최대 6개까지 추가할 수 있어요." Toast를 노출한다.
2
비수강생 진입 처리
  • 비수강생 진입 시 수강생 전용 메뉴는 잠금 상태로 노출한다.
  • 잠금 항목 클릭 시 "수강생 전용 메뉴입니다." Toast를 노출한다.
2-1 게스트 로그인 유도
  • 게스트 상태에서 수강생 전용 항목 클릭 시 로그인 유도 Modal을 노출한다.
  • [로그인하기] 버튼 클릭 시 로그인 페이지로 이동한다.
  • [닫기] 버튼 또는 배경 탭 시 Modal을 닫는다.
